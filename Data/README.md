# Conway's Game of Life

Реализация игры Жизнь Конвея на C# с расширенными возможностями для исследования клеточных автоматов.

## Возможности

- Полная реализация правил игры Жизнь
- Загрузка и сохранение состояний в JSON
- Настройки через JSON-файл
- Исследование стабильности с разной плотностью
- Классификация фигур (блоки, планеры, мигалки, etc.)
- Визуализация в консоли
- Множество предустановленных фигур
