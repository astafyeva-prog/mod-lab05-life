# Игра Жизнь Конвея

Реализация клеточного автомата "Игра Жизнь" с дополнительными возможностями.

## Функциональность

- Симуляция игры Жизнь Конвея
- Загрузка и сохранение состояний в JSON
- Настраиваемые параметры через settings.json
- Классификация фигур (блоки, ульи, планеры и т.д.)
- Исследование стабильности при разной плотности
- Автоматические тесты (16 тестов)

## Использование

1. Запустите приложение
2. Выберите режим:
   - Случайное заполнение
   - Загрузка предустановленных фигур
   - Исследование стабильности
   - Запуск с параметрами

## Управление

- ESC - выход
- SPACE - шаг вперед (в ручном режиме)

## Тестирование

Запуск тестов: `dotnet test`

## Структура

- `Life/` - основной проект
- `Life.Tests/` - модульные тесты
- `Data/` - результаты исследований
- `settings.json` - конфигурация
